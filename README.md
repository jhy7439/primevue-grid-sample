# Vue Grid Playground

Vue 3에서 Vuetify `v-data-table`과 PrimeVue `DataTable`을 같은 데이터로 비교하는 학습 프로젝트입니다.

## 실행

Node.js 20 이상을 설치한 뒤:

```bash
npm install
npm run dev
```

## 연습 포인트

- 그리드 전환 후 검색, 정렬, 페이지네이션 비교
- 체크박스 행 선택 상태 비교
- 슬롯(template)을 이용한 상태 태그와 작업 버튼 렌더링
- 직원 추가·수정·삭제에 따른 반응형 데이터 갱신

데이터는 새로고침 시 초기화되는 프론트엔드 실습용 샘플입니다.
